Data from PokeApi API: https://pokeapi.co/ . Git: https://github.com/PokeAPI/pokeapi

# Running the app

On your terminal: 

1. Install node modules
npm i

2. Run the backend service
npm run serve

3. Open the URL shown in the terminal on your web browser
